# Sandbox

## 定位

- `sandbox` 当前只是执行环境占位模型。
- 它表达的是“Worker 将来应该持有一个隔离执行容器”，不是当前已有完整沙箱能力。

## 边界

- 现在只提供占位类型 `DummySandboxContainer`。
- 当前不承担权限隔离、文件隔离、进程隔离等真实职责。

## 关键概念

- `DummySandboxContainer`
  当前 Worker 持有的占位对象。

## 核心能力

- 当前唯一能力是明确告诉外部：沙箱能力尚未接入。

## 使用方式

- 现在不要把真实安全边界假设建立在这个模块上。
- 后续如果接入真实沙箱，应替换这个占位层，而不是在别处偷偷做半套隔离。
